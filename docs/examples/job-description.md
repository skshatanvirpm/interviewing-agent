# Example Job Description

This fictional role specification is provided as a version-safe input for local development, testing, and demonstration.

## Role

Senior Machine Learning Engineer

## Company

Example Commerce

## Product context

Example Commerce operates a large marketplace where customers discover products, receive personalized recommendations, and complete digital transactions. The machine learning platform supports search, recommendation, risk signals, customer support automation, and experimentation.

## Responsibilities

- Design, train, evaluate, and deploy machine learning systems.
- Build reliable data and feature pipelines for offline and online use.
- Develop production GenAI and retrieval-augmented applications.
- Improve recommendation and ranking quality.
- Define offline evaluation, online experiments, and monitoring.
- Partner with product, data, platform, and risk teams.
- Review architecture and mentor engineers.
- Translate ambiguous product problems into measurable ML objectives.

## Required experience

- Strong Python and applied machine learning fundamentals.
- Experience with PyTorch, TensorFlow, or scikit-learn.
- Experience deploying and monitoring production models.
- Knowledge of cloud infrastructure and distributed data processing.
- Ability to reason about latency, reliability, cost, privacy, and failure modes.
- Clear communication and cross-functional collaboration.

## Preferred experience

- Transformer architectures and foundation-model applications.
- Retrieval-augmented generation, embeddings, reranking, and evaluation.
- Supervised fine-tuning, preference optimization, LoRA, or QLoRA.
- LLM guardrails and output validation.
- Recommendation systems, candidate generation, ranking, and bandits.
- Agentic workflows, tool use, memory, and orchestration.
- A/B testing, causal reasoning, calibration, fairness, and robustness.
- Technical leadership in ambiguous environments.

## Interview focus

- End-to-end project architecture and individual ownership.
- Technical decisions and rejected alternatives.
- ML fundamentals and factual correctness.
- Evaluation design and metric selection.
- Production deployment, monitoring, and failure recovery.
- Recommendation, GenAI, and retrieval trade-offs.
- Collaboration, disagreement handling, and judgment.

## Seniority

Senior or lead-level scope, with an expectation of independent technical ownership and cross-functional influence.
